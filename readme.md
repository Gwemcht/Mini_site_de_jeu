# 🎮 Mini sie de jeu

Bienvenue sur **Mini sie de jeu**, un site web regroupant une collection de jeux amusants et interactifs accessibles directement depuis votre navigateur !

## 🚀 À propos du projet

Ce projet a pour objectif de proposer une plateforme simple et divertissante où les utilisateurs peuvent jouer à différents mini-jeux sans installation.
